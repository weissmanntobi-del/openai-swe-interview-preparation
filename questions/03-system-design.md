# Backend and Platform System Design

## 19. Event Ingestion Platform

Design a high-volume event ingestion and processing platform.

Cover schemas, validation, ordering, partitioning, replay, quotas, backpressure, and data-quality observability.

## 20. Reliable Webhook Platform

Design multi-tenant webhook delivery.

Cover signing, secret rotation, retries, fairness, customer endpoint health, audit history, and abuse prevention.

## 21. Background Job Platform

Design a platform for long-running jobs.

Cover job states, leases, cancellation, retries, progress, prioritization, worker capabilities, and recovery.

## 22. Developer Build Service

Design a remote build and test platform.

Cover source isolation, caching, reproducibility, sandboxing, scheduling, artifact retention, and developer feedback.

## 23. Global Feature Policy Service

Design low-latency policy distribution with safe rollback.

Cover versioning, regional caches, propagation, auditability, authorization, and outage behavior.

## 24. Conversational Backend

Design a globally available conversation service with streaming responses and history.

Cover APIs, persistence, ordering, reconnect, model invocation, privacy, moderation, and cost controls.

## 25. Metrics Ingestion and Query

Design a high-cardinality metrics system.

Cover ingestion, aggregation, storage tiers, query latency, retention, cardinality control, and downsampling.

## 26. Resumable File Processing

Users upload very large files that trigger asynchronous processing.

Cover multipart upload, integrity, state machine, retries, malware scanning, progress, cancellation, and cleanup.

## 27. Configuration Rollout Platform

Design staged configuration rollouts to a large fleet.

Cover targeting, canaries, validation, rollout control, rollback, health gates, and auditing.

## 28. Enterprise Connector Platform

Design connectors that synchronize data from external enterprise systems.

Cover credentials, permissions, incremental sync, rate limits, retries, deletion, schema drift, and tenant isolation.
